Transfer Object variable naming convention.

Noncompliant Code Example

	public class TONameVerify {

	boolean booleanVar;
	byte byteVar;
	short shortVar;
	long longVarTO;
	
}

Compliant Solution

	public class TONameVerify {

	boolean booleanVar;
	byte byteVar;
	short shortVar;
	long longVar;
	Employee empTO;
}
